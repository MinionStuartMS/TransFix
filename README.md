# TransFix — Keyboard Layout Fixer & Translator

![image (4)](https://github.com/user-attachments/assets/838aec55-49ad-46a8-869e-1d3bd8f2f604)


**TransFix** is a handy desktop application for Windows that solves two common problems:
1. **Fixing keyboard layout** — when you forgot to switch the language and typed "ghbdtn" instead of "привет"
2. **Quick text translation** between English and Russian

The app features a stylish dark/light theme and works directly from the system tray.

## ✨ Features

- 🔄 **Layout Fixing** — automatically converts text typed in the wrong keyboard layout
- 🌐 **Translator** — translate between English and Russian (via Google Translate)
- 🎨 **Theme Customization** — personalize interface colors to your liking
- 📋 **Clipboard Integration** — quick paste and copy functionality
- ⚡ **Hotkeys** — `Ctrl+Enter` for fast processing
- 💾 **Settings Persistence** — your theme preferences are saved between sessions
